# movie-clustering-dashboard
# 🎬 Movie Clustering Dashboard using K-Means

An interactive Machine Learning dashboard built with Python and Streamlit that clusters movies based on ratings and popularity using the K-Means clustering algorithm.

---

# 📌 Project Overview

This project uses the MovieLens dataset to analyze movie ratings and automatically group movies into clusters using Unsupervised Machine Learning.

The dashboard allows users to:

* Select the number of clusters dynamically
* Visualize movie groups
* Analyze centroid positions
* Explore movies inside each cluster

# ⚙️ Installation

## 1. Clone the Repository

```bash
git clone https://github.com/your-username/movie-clustering-dashboard.git
cd movie-clustering-dashboard
```

## 2. Install Dependencies

```bash
pip install -r requirements.txt
```

## 3. Run the Streamlit App

```bash
streamlit run app.py
```

---

# 📊 How It Works

## Step 1: Data Loading

The system loads movie and rating datasets.

## Step 2: Data Processing

It calculates:

* Average rating per movie
* Total ratings count

## Step 3: Feature Scaling

Features are normalized using StandardScaler.

## Step 4: K-Means Clustering

Movies are grouped into clusters using K-Means.

## Step 5: Visualization

Clusters and centroids are displayed using scatter plots.
